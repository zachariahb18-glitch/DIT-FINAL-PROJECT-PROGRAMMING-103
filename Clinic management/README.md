# Clinic Management System

A professional desktop clinic management application built with Python, Tkinter, SQLite, and Matplotlib.

## Features
- Secure login with hashed passwords
- Dashboard with stats and charts
- Patient, doctor, appointment, medicine, billing, and report modules
- SQLite-backed database with automatic initialization and sample data
- CSV/PDF export support
- Backup and restore support

## Run
```bash
python main.py
```
